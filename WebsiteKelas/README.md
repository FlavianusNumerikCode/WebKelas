# WebKelasXI

Website komunitas kelas XI [nama angkatan], privat untuk anggota kelas.

## Fitur:
- Home/banner, profil, jadwal, galeri, kuis mingguan (dengan poin dan level), hall of fame, panel admin.
- Panel admin dapat kelola soal kuis, hasil kuis, poin siswa, foto galeri, jadwal pelajaran.
- Tampilan modern, responsif, animasi halus.

## Teknologi
- Frontend: HTML, CSS, JavaScript (animasi & interaksi)
- Backend: PHP, MySQL
- Jalankan secara lokal via XAMPP / hosting PHP+MySQL
